# Medisign

## Description
Medisign is a project aimed at providing a robust solution for managing medical records and signatures digitally. It leverages blockchain technology to ensure the integrity and confidentiality of sensitive patient information.

## Installation Instructions
To get started with Medisign, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/h3-maneesha/medisign.git
   ```
2. Navigate into the project directory:
   ```bash
   cd medisign
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage
To start using the application, run the following command:
```bash
npm start
```
Access the application in your web browser at `http://localhost:3000`.
